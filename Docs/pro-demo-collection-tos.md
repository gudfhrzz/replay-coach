## 결론 요약

| 소스 | 판정 | 근거 |
|------|------|------|
| **HLTV** | ❌ 자동 수집 금지 | ToS가 스크래핑·데이터마이닝 명시 금지 + 경쟁 제품 구축 금지 |

## 1. HLTV — 자동 수집 부적합 (제외)

- ToS(hltv.org/terms)가 **"데이터 마이닝·웹 스크래핑을 수행·조력·조직하는 것"을 금지**하고,
  **"유사하거나 경쟁하는 웹사이트·제품·서비스 구축" 금지** 조항도 있음 — 우리 제품은
  후자에도 걸릴 소지가 있다.
- 기술적으로도 Cloudflare로 봇 접근을 차단 중(약관 페이지 fetch 자체가 403).
  robots.txt도 stats/results 필터 경로 다수 Disallow.
- → **HLTV에서 자동 데모 수집은 하지 않는다.** 통계 대조용 수동 열람만.

## 2. FACEIT — 공식 Downloads API (권장 경로)

- 공식 개발자 프로그램 존재 (docs.faceit.com). Data API 무료, 레이트리밋 10,000 콜/시간.
- 데모 파일 URL은 Data API 매치 엔드포인트에 있지만 **실제 다운로드는 별도
  Downloads API 승인 필요**:
  - 신청서: https://fce.gg/downloads-api-application (유스케이스 설명 3개 섹션)
  - 심사 기간 **약 30일** → 리드타임이 기니 조기 제출 권장
  - 승인 시 전용 액세스 토큰으로 signed URL 발급받아 .dem.gz 다운로드
- FACEIT엔 FPL 등 프로·준프로 리그가 돌므로 프로 데이터 소스로 성립.
- ⚠ **개발자 ToS 전문(developers.faceit.com/terms)은 Cloudflare 챌린지 때문에
  자동 열람 불가.** 신청서 제출 전에 브라우저로 열어 상업적 이용·데이터 보관·
  파생물(통계 모델) 조항을 확인할 것. 어차피 신청서에 유스케이스를 밝히므로
  승인 자체가 명시적 허가로 기능한다.

## 3. ESTA 데이터셋 — 프로토타이핑용 즉시 사용 가능

- **ESTA (Esports Trajectories and Actions)**: 프로 CS:GO 경기 1,558개
  (Online 878 + LAN 680, 2021-01~2022-05)를 awpy로 파싱한 공개 데이터셋.
  github.com/pnxenopoulos/esta, 논문 arXiv:2209.09861
- **라이선스 CC BY-SA 4.0** — 출처 표기 + 파생물 동일 라이선스 조건으로 사용 가능.
- 형태: 원본 .dem이 아니라 **파싱된 JSON**(.xz, 라운드 메타데이터 + 2Hz 프레임).
  awpy 파싱 결과라 라운드별 경제 필드(장비 가치, buy type)가 포함된 것으로 알려짐
  — 다운로드 후 스키마 검증 필요.
- 한계: **CS2가 아니라 CS:GO** (경제 상수 다름: 2026 현재 CS2는 MR12, CS:GO는 MR15).
  프로 분포 모델의 *방법론 검증용*으로 쓰고, 실서비스 분포는 FACEIT CS2 데모로 재구축.
- ⚠ SA(동일조건) 조항: ESTA 파생 *데이터셋을 재배포*하면 CC BY-SA로 공개해야 함.
  내부 학습·통계 서비스 제공에 미치는 범위는 상용화 전 법률 검토 항목.

## 4. Valve — .dem 파일 자체의 권리 ⚠

- 데모 파일은 Valve 게임 콘텐츠의 기록물. FACEIT/토너먼트 주최사가 배포권을 갖더라도
  Valve 측 약관(Steam 구독자 계약 등)이 상업적 파생 이용에 뭐라 하는지는 별도 확인 필요.
- 커뮤니티 도구(HLTV, csstats.gg, Leetify 등)가 광범위하게 상용 운영 중인 것을 보면
  관행상 허용 영역이나, 상용화 전 확인 목록에 올려둔다.
